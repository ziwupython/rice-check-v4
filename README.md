# rice-check-v4
【2026毕业设计】水稻植物病害识别系统~Python+深度学习+人工智能+模型训练+图像识别

## 一、介绍
水稻植物病害识别系统（['细菌性枯萎病(Bacterial blight)', '稻瘟病(Blast)', '褐斑病(Brown spot)', '东格鲁病毒病(Tungro)']），技术栈如下：
● 前端：Vue3+Element plus
● 后端：Flask框架
● 算法：卷积神经网络算法（resnet50）、TensorFlow
关键技术：深度学习、模型训练、图像识别、人工智能、Python、CNN卷积网络
